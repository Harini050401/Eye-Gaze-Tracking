# Eye-Gaze-Tracking

## Description:
This project allows users to control the mouse using eye movements. It uses a webcam to track eye position and perform actions like cursor movement and clicking.

## Features:
- Eye-based cursor movement
- Automatic click using hover
- Voice feedback before clicking
- Visual progress indicator

## Technologies Used:
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Tkinter
- pyttsx3

## How to Run:
1. Install required libraries:
         pip install opencv-python, mediapipe, pyautogui, numpy, pyttsx3
2. Run the file:
        Python file.py

 ## Usage:
- Move your eyes to control cursor
- Keep cursor steady to perform click
- Press 'Q' to exit
